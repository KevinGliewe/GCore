Namespace GCore.XConsole.ConsoleBrowserObjects

Classes

ConsoleForm

The root object of the hierarchy. Contains enough information to render labels (read-only) and textboxes (read/write) to a console screen. It manages interaction with the keyboard to solicit input to fields, navigation through the fields, and completion of the form.

FormCompleteEventArgs

A class used to communicate the results of a FormComplete event to subscribed clients. More importantly, it allows the subscribed client to communicate back to the ConsoleForm whether or not the old form should be disposed or not.

KeyPressEventArgs

Class to provide parameters to subscribers to KeyPressed events.

Label

A class used to store the data necessary for a write-only object rendered on the console screen.

Labels

A collection class of Label objects.

Line

Representation of a vertical or horizontal line on the console.

Lines

Collection class of Line objects.

Point

An object to use to specify the X and Y coordinates of an object on the console screen. A Point object already exists in the framework, but it's hidden in System.Windows.Forms.Drawing, which I didn't want to include here.

StdConsoleObject

The base class for textual information displayed by the console.

Textbox

An object used to manage the data necessary to render read-write objects to the screen.

Textboxes

A collection class of Textbox objects.

Enums

Line.LineOrientation

An enumeration to describe the direction of the line to be drawn, from the origin.

Delegates

ConsoleForm.onFormCancelled

Represents the method that will be invoked when a form is cancelled.

ConsoleForm.onFormComplete

Represents the method that will be invoked when a form is submitted.

ConsoleForm.onKeyPress

A delegate to a method that handles the event raised when a key is pressed on a ConsoleForm.
