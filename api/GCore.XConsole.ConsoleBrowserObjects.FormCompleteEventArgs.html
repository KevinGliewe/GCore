Class FormCompleteEventArgs

A class used to communicate the results of a FormComplete event to subscribed clients. More importantly, it allows the subscribed client to communicate back to the ConsoleForm whether or not the old form should be disposed or not.

Cancel

Property to set to prevent the ending of the keypress loop, and the disposal of form resources. If true, the form will not be disposed and the keypress loop thread will continue to wait for keypresses. If false (default), the form will be disposed at the end of the FormComplete event handler.
