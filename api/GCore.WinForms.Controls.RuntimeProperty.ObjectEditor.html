Class ObjectEditor

Singleton class that takes care of showing the Runtime ObjectEditor form with it's window finder. To use this you have to enable it:

RuntimeObjectEditor.ObjectEditor.Instance.Enable();

The default shortcut key used it "Control+Shift+R". If you want to use a different shortcut, change the HotKey
